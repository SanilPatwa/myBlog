A lightweight blog creation web application with JWT-based authentication. Built with an Express.js backend and a plain HTML/CSS frontend — no database required, no frontend frameworks.
Features

Create, view, and manage blog posts
JWT-based user authentication (login / register)
In-memory or file-based data storage (no DB setup needed)
Clean HTML/CSS frontend — no frameworks
RESTful API with Express.js
Protected routes via JWT middleware

Tech Stack

Backend: Node.js, Express.js, jsonwebtoken, bcryptjs
Frontend: HTML, CSS, Vanilla JavaScript
Auth: JSON Web Tokens (JWT)
Storage: In-memory / JSON file (no external database)
